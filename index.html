¿Cómo funciona?

  1. Selecciona un archivo CSV usando el botón "Seleccionar Archivo" o arrastra el archivo al área indicada
  2. La aplicación detectará automáticamente el delimitador (coma, punto y coma, etc.)
  3. Las siguientes columnas se eliminarán automáticamente: 3,5,7,9,11,13,15,17,21,23,25,27,29,31,33,35,37,39,41
  4. Si la detección no es correcta, puedes seleccionar manualmente el delimitador
  5. Cuando hayas terminado, haz clic en "Exportar a CSV" para descargar el archivo modificado
  6. Usa "Restablecer" para comenzar con un nuevo archivo
